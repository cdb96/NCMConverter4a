# NCMConverter4a
---
一个适用于安卓的歌曲转换器
### 使用方法：
1.从本软件右下角按钮选择文件，进行转换
2.从外部文件管理器批量选择文件分享至该软件，进行转换
### 注意事项：
本项目仅用于学习用途，请确保您已充分了解相关版权协议，否则请勿使用。
