# NCMConverter4a
---
一个适用于安卓的歌曲转换器
### 使用方法：
1.从右下角按钮选择文件 

2.从外部文件管理器选择文件分享至该软件

PS:上述方法均支持批量选择文件
### 设置说明：
1.原始写入模式：开启后软件仅进行解密，不进行写入元数据操作（只适用于NCM文件，因为KGM文件根本没提供元数据）

2.线程数滑条：调整转换时使用的线程数量
